# Goomti Sales Tracker — React App

Daily sales performance monitor built with React,
using real restaurant data from Goomti Restaurant.

## 🔗 Live Demo
[View Sales Tracker](https://goomti-react-sales-tracker-a728.vercel.app/)

## Features
- **Week filter** — filter data by week using toggle buttons
- **Area chart** — daily revenue trend with 7-day rolling average
- **Hourly chart** — sales by hour with 10pm delivery spike highlighted
- **Weekend vs Weekday** — visual revenue split with progress bars
- **Daily log table** — every trading day with vs-average comparison

## Tech Stack
React · Recharts · PapaParse · JavaScript
